# Vezir Archive

Historical artifacts archived from [vezir](https://github.com/ahmetcagriakca/vezir) main repository.

## Contents

- `sprints/` — closed sprint folders (sprint-12 through sprint-74, legacy sprint-33 through sprint-41)
- `evidence/` — historical sprint evidence packets (sprint-8 through sprint-74)
- `stale/` — quarantined stale documents
- `stale-ai-docs/` — archived AI governance docs
- `stale-handoffs/` — archived handoff documents
- `handoffs/` — historical handoff archive
- `phase-reports/` — historical phase closure and sprint reports
- `process-history/` — archived process documentation
- `review-packets/` — finalized review packets
- `debt-plans/` — archived debt reduction plans
- `phase1-tasks/` — Phase 1 task artifacts
- `misc/` — session reports, reorganization reports, loose sprint artifacts

## Policy

- This repo is append-only archive. No deletions without operator approval.
- Active governance docs remain in the main vezir repo.
- This repo has no CI/CD, no runtime role, no tooling dependencies.

## Source

Archived per D-113 (Archive Boundary Decision) from the Vezir project.
