# Sprint 19 — Task Breakdown

**Sprint:** 19
**Phase:** 6
**Title:** Single-Repo Automation MVP
**Model:** A

---

## Track 1: Foundation

**19.1 — plan.yaml schema freeze**
Create `docs/sprints/sprint-19/plan.yaml` with full schema. Include: sprint metadata, authority block, issue config, all tasks with IDs/titles/types/branch_names, validation flags.

**19.2 — plan.yaml task breakdown validator**
Script: `tools/validate-plan-sync.py`. Reads plan.yaml, reads task breakdown doc, checks: task IDs match, task titles match, no orphans in either direction. Fails on conflict.

## Track 2: Issue Automation

**19.3 — issue-from-plan.yml workflow**
GitHub Actions workflow. Trigger: workflow_dispatch. Reads plan.yaml, creates parent sprint issue, creates child task issues. Generates `issues.json` mapping. Idempotent.

**19.4 — issues.json schema and mapping**
Generated by 19.3 workflow. Schema: `{ sprint, parent_issue, tasks: { "19.1": { issue, branch, pr } } }`. PR field null until PR exists.

## Track 3: Branch Discipline

**19.5 — Branch naming contract and check script**
Document: `docs/shared/BRANCH-CONTRACT.md`. Script: `tools/check-branch-name.sh`. Validates current branch matches sprint naming pattern.

**19.6 — main protection verification**
Verify GitHub branch protection is active. Test: attempt direct push to main fails. Document protection config.

## Track 4: Process Alignment

**19.7 — Governance update**
Create `docs/shared/GOVERNANCE.md` with all governance rules: reviews advisory, operator sole authority, branch-per-task mandatory, manual merge after gate PASS.

## Gates

**19.G1 — Mid Review Gate**
After Track 1 + Track 2 complete. Review: plan.yaml validates, issues created, validator works. Branch-exempt.

**19.G2 — Final Review Gate**
After all tracks complete. Full sprint evidence review. Branch-exempt.

**19.RETRO — Retrospective**
Must answer: Did automation reduce manual work? Is branch-per-task working? Should PR-per-task be activated?

**19.CLOSURE — Sprint Closure**
All task branches merged to main. All evidence files present. Operator sign-off.
